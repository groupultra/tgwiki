# Membuat Voting

## Gambaran Fitur

Klien resmi Telegram sekarang memiliki fitur "voting" yang terintegrasi, mendukung tampilan real-time jumlah suara dan persentase untuk setiap pilihan, membuat keputusan kelompok lebih mudah dan intuitif.

## Tips Ramah

Saat mengaktifkan **voting anonim**, identitas semua peserta tidak akan dipublikasikan.

## Penjelasan Operasi Fitur

### Tindakan yang Dapat Dilakukan oleh Pencipta Voting

- Memulai voting baru
- Menggunakan **tekan lama (mobile)/klik kanan (desktop)** untuk menghentikan voting saat ini
- Juga melalui tekan lama atau klik kanan, dapat membatalkan pilihan saat ini dan mengatur voting kembali

### Tindakan yang Dapat Dilakukan oleh Peserta

- Cukup sentuh salah satu pilihan untuk menyelesaikan voting
- Jika ingin mengubah pilihan, tekan lama atau klik kanan pada pilihan saat ini, batalkan kemudian voting ulang

## Cara Membuat Voting

### Telegram iOS

Klik **ikon penjepit di kiri bawah** → Pilih **"Voting (Poll)"**

### Telegram / Telegram X Android

Klik **ikon penjepit di kanan bawah** → Pilih **"Voting (Poll)"**

Penjelasan antarmuka:

![Penjelasan Antarmuka Voting](/markdown/img-3.png)

### Telegram macOS

Arahkan **mouse ke atas ikon penjepit di kiri bawah** → Pilih **"Voting (Poll)"**

### Klien Desktop Windows / macOS / Linux

Klik **"menu tiga titik" di kanan atas** → Pilih **"Buat Voting (Create poll)"**

## Catatan Penting

Jika tidak menemukan fungsi "voting", atau sistem memberi tahu "**Versi Telegram Anda tidak mendukung pesan ini (This message is not supported by your version of Telegram)**", silakan periksa apakah perlu memperbarui aplikasi ke versi terbaru.
