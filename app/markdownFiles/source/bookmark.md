# 书签

## 功能特点

- 支持多终端、多设备实时同步
- 文件存储数量不限，支持各种内容类型，但单个文件大小上限为 2000MB
- 无论是私聊、群组还是频道消息，都能轻松转存到"收藏夹"，方式为：**转发 → 收藏夹**

## 开启方法

所有设备客户端都可以在聊天输入框中搜索你自己的"用户名"，即可找到"收藏夹"。

### iOS 端

进入 **设置 → 收藏夹**

### Android 端

点击界面**左上角的"三条横线"图标** → **收藏夹**

### macOS 端

- 点击**顶部搜索框** → **选择"收藏夹"**
- 快捷键：`Ctrl+0`

### Desktop 端

- 点击**左上角"三条横线"** → 再点击你的**头像或收藏夹图标**
- 快捷键：`Ctrl+0`
